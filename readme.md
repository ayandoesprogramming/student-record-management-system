# Student Record Management System (C)

A menu-driven Student Record Management System built in C.

## Features

* Add student
* Search student by roll number
* Edit student details
* Delete student
* Display all students
* Data saved to file (persistent storage)

## Concepts Used

* Structures
* Functions
* File handling
* Arrays
* Menu-driven programming

## How to Run

Compile:
gcc main.c -o srms

Run:
./srms
